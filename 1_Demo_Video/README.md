# Demo video

`BallDarkEcho_Final_Demo.mp4` is the final gameplay demonstration for the submitted build.

The `Progress` directory contains the development recordings from 1, 3, 8, 15, and 17 June 2026, including alternate recordings captured on the same day.

All MP4 files are managed by Git LFS. If a file appears as a text pointer after cloning the repository, install Git LFS and run `git lfs pull`.
