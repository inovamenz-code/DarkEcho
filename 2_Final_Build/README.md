# Final Windows build

1. Install Git LFS and clone the repository with LFS content enabled.
2. Extract `BallDarkEcho_Windows_Development.zip`.
3. Run `Windows/BallDarkEcho.exe`.

The archive excludes editor caches, PDB debug symbols, runtime logs, and local development-only output. It retains the packaged Unreal runtime, cooked game content, and Wwise runtime files required by the build.
